# revq

## Prerequisite
1. Install rust
```
curl https://sh.rustup.rs -sSf | sh
```

## Getting started
1. Clone repository and cd to it.
```
git clone https://github.com/k3ii/revq
```
2. Install the package via cargo
```
cargo install --path .
```
3. Initialize revq
```
revq init
```
4. Find bugs =D
